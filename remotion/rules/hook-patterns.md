# 开场钩子设计指南

开场 3-8 秒决定观众是否继续看。钩子不是噱头，是让观众感受到"这个视频和我有关"。

## 5 种钩子类型

### 1. 痛点唤醒
**公式**：你是不是也…… + 痛点描述
**效果**：瞬间共鸣，让人觉得"就是在说我"
**适用**：工具类、方法类、解决具体问题的视频

```
"你是不是也总觉得学了很多，但脑子里还是一片空白？"
"你是不是每次开会都不知道说什么，然后后悔自己没开口？"
"用了 ChatGPT 半年，你有没有发现它越来越废？"
```

**视觉**：大字居中，逐字出现，背景深色，无其他装饰

---

### 2. 反常识
**公式**：你以为…… 其实……
**效果**：制造认知冲突，让人好奇"为什么"
**适用**：知识解读、认知升级、颠覆常识的内容

```
"你以为价值投资就是买好公司长期持有——错了。"
"你以为 AI 会抢走你的工作——真正危险的是另一件事。"
"你以为多读书就能变聪明——数据不这么说。"
```

**视觉**：两段式——"你以为"用普通字体，"其实/错了"用强调色大字弹出

---

### 3. 承诺结果
**公式**：X 分钟让你…… / 看完这个你就能……
**效果**：直接告诉观众回报，降低观看成本感
**适用**：教程类、技能类、有明确产出的内容

```
"3 分钟，让你彻底搞懂 RAG 是什么。"
"看完这个，你下次开会就有话说了。"
"60 秒，告诉你巴菲特真正的买入逻辑。"
```

**视觉**：数字要大（120px+），承诺结果用强调色，动画用 bouncy spring

---

### 4. 场景代入
**公式**：想象一下…… / 假如你……
**效果**：把观众放进故事里，情感代入
**适用**：故事类、案例类、有场景感的内容

```
"想象你刚买了一只股票，第二天暴跌 30%……"
"假如你明天要给 CEO 汇报，但 PPT 还没做……"
"你正在面试，面试官问了你一个你从没想过的问题……"
```

**视觉**：文字逐句出现，配合场景色调（暖色=日常，冷色=危机感）

---

### 5. 数据冲击
**公式**：X% 的人…… / 一个数字 + 颠覆感
**效果**：用数据制造震撼，让人想知道"为什么"
**适用**：研究类、趋势类、有数据支撑的内容

```
"92% 的人学英语失败——不是因为懒，是因为方法错了。"
"过去 10 年，做指数基金的人跑赢了 85% 的基金经理。"
"1000 块的比特币，如果你 2013 年买了，现在是 6000 万。"
```

**视觉**：数字超大（120-160px），先出数字，再出解释文字，用 NumberCounter 组件

---

## 选择钩子的规则

| 内容类型 | 推荐钩子 | 原因 |
|---------|---------|------|
| 概念解释（RAG/区块链） | 痛点唤醒 / 反常识 | 让人感受到"这个概念和我有关" |
| 技术原理（工作原理） | 场景代入 / 承诺结果 | 降低门槛感，给人学习动力 |
| 投资/商业分析 | 数据冲击 / 反常识 | 用数据建立可信度 |
| 方法论/技能 | 承诺结果 / 痛点唤醒 | 明确回报，让人觉得值得看 |
| 故事/案例 | 场景代入 | 情感代入最强 |

## 钩子质量检查

- [ ] 钩子是否在 3-8 秒内能说完？（不要超过 30 字）
- [ ] 是否让观众感受到"这和我有关"？
- [ ] 是否制造了"然后呢"的好奇心？
- [ ] 视觉上是否简洁有冲击力（不超过 2 个文字元素）？

## Step 间的小钩子（保持节奏）

每个 Step 结尾，要给下一 Step 留钩子：

```
Step 1 → Step 2: "但大多数人都走错了一步……"
Step 2 → Step 3: "但有一个人不这么看。"
Step 3 → Step 4: "他是怎么做到的？"
Step 4 → Step 5: "结果呢？"
Step 末尾:       "所以下次遇到这种情况，你可以……"
```
