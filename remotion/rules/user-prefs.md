# 用户偏好记忆规则

## 启动时检查

每次开始 Remotion 创作任务，先用 Glob 工具检查是否存在：
```
{workspace}/.claude/projects/*/memory/remotion_user_prefs.md
~/.claude/projects/*/memory/remotion_user_prefs.md
```
找到则读取，在本次创作中优先应用，无需再询问风格。

## 何时收集偏好

| 触发情景 | 记录内容 |
|---------|---------|
| 用户确认或说「我喜欢 X 风格」 | 风格体系偏好 |
| 用户反复选同一配色 | 主色/强调色 HEX |
| 用户指定字体 | 偏好字体名 |
| 用户说「不要 X」「太 X 了」 | 禁忌项 |
| 用户说明内容领域 | 教育/技术/创意/商务等 |
| 用户说明目标平台 | B站/小红书/YouTube/视频号 |
| 用户指定视频比例 | 竖屏/横屏/方形 |

## 保存格式

用 Write 工具存入当前 project memory 目录，文件名 `remotion_user_prefs.md`：

```markdown
---
name: remotion-user-preferences
type: user
description: 用户 Remotion 视频创作风格与偏好记录
---

## 创作信息
- 内容领域：
- 目标平台：
- 常用比例：

- 风格体系：[暗色专业系 / 温暖小红书系]
- 偏好主题：
- 主色：
- 强调色：
- 偏好字体：

## 动效偏好
- 节奏感：[轻盈/沉稳/弹跳感强]
- 转场风格：

## 禁忌项
-

## 更新记录
- [日期] 首次创建
```

## 应用规则

- 自动选对应风格，不重复展示全部选项
- 禁忌项加入本次任务黑名单
- 每次更新偏好后，在文件末尾追加更新记录行
